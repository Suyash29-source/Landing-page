# NexusChat Landing Page

A modern, animated landing page built with **Next.js**, **Tailwind CSS**, **Framer Motion**, **GSAP + ScrollTrigger**, and **Lenis** smooth scrolling.

## Project Structure

```text
app/
  globals.css
  layout.tsx
  page.tsx
components/
  Hero.tsx
  Features.tsx
  ScrollDemo.tsx
  ChatPreview.tsx
  Community.tsx
  Stats.tsx
  CTA.tsx
  Footer.tsx
  SmoothScrollProvider.tsx
lib/
  animations.ts
```

## Run Locally

1. Install dependencies:

```bash
npm install
```

2. Start dev server:

```bash
npm run dev
```

3. Open:

```text
http://localhost:3000
```

## Production Build

```bash
npm run build
npm run start
```
